# Notes on AIllustrator

### ********************************
**Space** for moving object

Windows Tile

View > Outline

**D** : default

Transform (direct selection for Anchors)

### ********************************
Shape Builder = Pathfinder tool (merge, minus)

Expand option: turn strokes into fills.

Right-click : Arrange > Bring to front

### ********************************

Coupmound path: when strokes within strokes

### ********************************

Object > Lock Selection (making it easier to select other object)

Shift click to add or remove from Selection

Select > Save selection

Magic Wand (selecting same coloured objects)

Lasso Tools (to select objects by drawing), then switch back to black arrow, else...

Select than Inverse (Warning, select other artboards)

### ********************************

Start workong professionnally with STRUCTURE 

Ctrl G : turning object into a GROUP, then in the layers ready to be renamed.

tip: Click sur la loupe dans layers pour retrouver un objet selectionné

### ********************************

Ne pas grouper parfois, pour conserver un mouvement séparé. Utiliser des layers pour "grouper".

Select, edit > cut, Create new layer, edit > Paste in Front.

Maintenant, on peut lock une layer pour agir sur une autre.

### ********************************

Isolation, by clicking twice on an group-object

### ********************************

Align and distribute objects

Select objects and look at the top bar icons.

Or in the Align Panel > options > Align to key objects > distribute spacing

### ********************************

Working with Guides

Smart Guides.

Drag your own guide from the ruler. Alt key for switching horiz/vert.

Unlock them in View > Guides > unlock

Set the x and y values if you wish

Turn Objet into guide: View > Guides > Make guide.

View > Show Grid + Snap to Grid

Setup the grid in preferences.

### ********************************

Artboard or Global Rulers ?

Right click on the Ruler.
