##### 1/ Puisqu'en java "tout est classe", l'élément fondamental de java est la classe.
  

##### 2/ Une classe est constituée d'attributs et de méthodes
  

##### 3/ différents types de variables en java: 
  

	* boolean
	** byte, short, int, long
	*** float, double
	**** char, string


##### 4/ En qualifiant un opérateur d'arithmétique s'il effectue des opérations arithm. sur ses opérandes, on trouve, en java, 6 opérateurs arithm : 
	= (affectation)
	+ (addition)
	- (soustraction)
	* (multiplication)
	/ (division)
	% (calcul du reste de la division entière)

##### 5/ Pour développer en java, sont strictement nécessaires: une JSE, un compilateur java, et un éditeur de texte. Le JRE devra aussi être installé si l'on a besoin d'exécuter le résultat de notre développement.
D'autres outils existent qui, quoique non proprement nécessaires, simplifieront le développement, à commencer par un IDE (facilitant l'écriture et le build).


##### 6/ L'API de Java, c'est l'ensemble des ressources (classes, fonctions) qui permettent de profiter des diverses fonctionnalités natives, autrement dit des services internes de la JavaSE (ou JavaEE). 
Précisons que l'API de Java est foisonnante, et propose notamment des services pour le développement d'UI, d'applications avec persistance des données, ou encore d'applications web.

##### 7/ on dispose en java de structures de contrôles: conditonnelles ou itératives.
		  if (à laquelle on ajoute éventuellement else), switch.
		  for, while.

		else, enum, extends
		false, final, finally, float, for
		if, implements, import, instanceof, int, interface
		long
		native
		new, null
		package, private, protected, public
		return
		short, static, super, switch, synchronized,
		this, throw, throws, transient, true, try
		void, volatile
		while

##### 9/  POO est un acronyme français employé pour signifier l'expression "Programmation Orientée Objet"

##### 10/ On peut avancer 3 concepts caractéristiques de la POO: l'encapsulation, l'héritage, le polymorphisme.
