### 5/Compilation et exécution, dans la CMD, d'un "Hello World"

fichier Test.java
```
public class Test {

    public static void main(String[] args){
    
        System.out.print("Hello World !");
    }
}
```

compilation
```
javac Test.java 
```

exécution
```
java Test  // Cette commande affiche "Hello World !"
```
